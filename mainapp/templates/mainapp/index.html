Bienvenido

Sistema de administracion de lacteos, donde se podran registrar recolectores, productores, pedidos de leche y ver las facturas de cada pedido.

Recolector

Sera el encargado de hacer las recolecciones de leche y para su registro se deben diligenciar unicamente los datos personales.

Productor

Sera quien haga un pedido de leche y para su registro se deben diligenciar sus datos personales y al menos una finca.

Finca

Cada finca debera estar relacionada con un productor y para registrarla se necesitan unicamente el nombre de la finca, direccion y telefono.

Facturas

Se podran visualizar las facturas en cada finca seleccionada segun los pedidos realizados.

Novedad

En cada pedido realizado se puede presentar una novedad la cual se va a ver reflejada al momento de gestionar las facturas

Modificar Datos

Se podra editar o eliminar los datos de los Recolectores, Productores y fincas.
